# About Python

1. check version of python
  - run: `python --version`

2. check version of package manager, pip, which is bundled with python3
  - run: `pip3 --version`

3. path to python and pip: 
  - ` /opt/homebrew/lib/python3.11`

4. upgrade pip: 
  - run: `python -m pip install --upgrade pip`
